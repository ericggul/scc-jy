# w1 Socket Contract

The `w1` experiment owns only these events and the `experiment:w1` room.

- `w1:join` from client to server with `{ role: "mobile" | "screen" }`.
- `w1:hello` from server to the joining client with socket id, presence, and last `w1` signal.
- `w1:presence` from server to `experiment:w1` clients only.
- `w1:signal:in` from mobile/screen clients to server.
- `w1:signal:out` from server to `experiment:w1` clients only.

Future experiments must define their own module and event prefix, for example
`w2:*`, and must not reuse `w1` state, rooms, or events.
