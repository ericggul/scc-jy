# Stock 2

Route: `/stock/2`

This variant is an interactive Bloomberg-inspired terminal built from distinct
function screens rather than a generic dashboard. It preserves command grammar,
dense hierarchy, tabular typography, numbered actions, and semantic color
without using Bloomberg branding.

The default `HOME` screen combines an equity monitor, selected-security
candlestick/volume study, market-moving news, breadth, and the next economic
release. Supported command tokens include a listed ticker plus `HOME`, `WEI`,
`GP`, `TOP`, or `ECO`. Selecting a security in `HOME` or `GP` updates the price
study. The fixed-format workstation scales its type, spacing, tables, charts,
and visibility rules with the outer frame.

Primary files:

- `components/dashboard/stock/2/index.tsx`
- `components/dashboard/stock/2/data.ts`
- `components/dashboard/stock/2/terminal.module.css`

The discarded prototype must not be restored. See the shared
[Bloomberg Terminal UI research](./research.md).
