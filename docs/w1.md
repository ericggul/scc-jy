# w1 Experiment

Routes:

- `/w1`
- `/w1/mobile`
- `/w1/screen`

Files:

- `app/w1/page.tsx`
- `app/w1/mobile/page.tsx`
- `app/w1/screen/page.tsx`
- `hooks/use-experiment-socket.ts`
- `socket/experiments/w1.mjs`
- `socket/experiments/w1.md`

Intent:

`w1` is a minimal multi-device Socket.IO experiment with isolated mobile and screen routes.

Socket contract:

- Room: `experiment:w1`
- Events:
  - `w1:join`
  - `w1:hello`
  - `w1:presence`
  - `w1:signal:in`
  - `w1:signal:out`

Rule:

`w1` state and events must not be shared with future socket experiments.
